Text2Video Tools Comparison, Ongoing


Category     Duration     Tool / Material    
ML/AI; experiment April 2024 - Present @ University of Washington Python

We will compare different Text2Video tools and models with a slightly different set of prompts.

In addition to the test with Text2Video-Zero, one of the DXARTS pretrained models collaborated with Juan Pampin will be used. The prompts for this experiment are not the same as the others since we have customized the prompt in our model. Please check the section Text2Video-Zero & DreamBooth.
