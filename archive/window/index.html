Blurry Window


Category     Duration     Tool / Material    
AI/ML Application; Design; Development 2021 @ New York University JavaScript

Cold rainy day... I use the handpose nodes to calculate the center of palm. When the model recognizes the user's hand, it changes the opacity of the circle drawn on the pixel point where the center of the hand is located. The sound of wiping glass can be heard along with the hand movement. This is achieved by calculating the hand displacement between frames.
